Vad sidan omfattar

Från enstaka objekt till större miljöer

Vår styrka är att se skillnaden mellan skrot och sådant som kan få ett nytt liv. Därför tittar vi inte bara på vad som ska bort, utan också på vad som kan repareras, säljas vidare, demonteras för delar eller sorteras som metall.

Återbruk före avfall

Vi försöker ta tillvara sådant som fortfarande har nytta, funktion eller reservdelsvärde innan det går vidare som skrot.

Vi behöver inte ta allt för att kunna hjälpa

Ett uppdrag kan handla om hela miljön, ett urval av objekt eller bara en bedömning av vad som är intressant. Vi är tydliga med vad vi kan ta hand om och vad som behöver lösas på annat sätt.

Överenskommelse

Vi bestämmer vad som ingår, eventuell ersättning, hämtning och praktiskt upplägg.

Det vi ofta tittar efter

Objekt med fortsatt värde

  • Traktorer, gräsklippare och andra maskiner.
  • Jordbruksredskap, släp och verkstadsutrustning.
  • Motorer, växellådor, reservdelar och verktyg.
  • Metall, kabelskrot och sorterade material.
  • Lagerinredning, pallställ och användbar utrustning.

Viktig avgränsning

Inte vanlig flytt- eller sophämtning

Vår huvudinriktning är sådant som kan köpas, återbrukas, demonteras eller sorteras som metall. Blandat hushållsavfall, byggsopor, farligt avfall och osorterade sopor ligger normalt utanför vår tjänst.

  • Vi lovar inte att ta hela innehållet.
  • Alla uppdrag bedöms individuellt.
  • Kostnad eller ersättning bestäms före start.

Vanliga frågor

Frågor om återbruk och tömning

Hjälper ni till att tömma en hel gård eller maskinhall?

Vi kan bedöma större miljöer, men vad vi tar hand om avgörs från fall till fall. Vår inriktning är främst maskiner, redskap, verktyg, metall, reservdelar och andra återbrukbara objekt.

Kan ni köpa vissa saker och lämna resten?

Ja. Ett upplägg kan bestå av att vi köper utvalda objekt, hämtar annat enligt överenskommelse och tydligt avgränsar det vi inte tar.

Måste allt vara sorterat innan ni kommer?

Inte nödvändigtvis, men tydliga bilder och en grov uppdelning mellan maskiner, metall, verktyg och övrigt gör bedömningen mycket enklare.

Gör ni platsbesök?

Vid större uppdrag kan ett platsbesök vara nödvändigt. Vi börjar vanligtvis med bilder och grundläggande information för att avgöra om ett besök är motiverat.

Tar ni även lantbruksredskap och gamla traktorer?

Ja, sådana objekt kan vara intressanta både fungerande, defekta och som reservdelsobjekt. Skicka bilder, modell om du vet och ort.

Börja med några bilder och en kort beskrivning

Visa platsen, objekten och omfattningen. Då kan vi snabbt säga om vi vill gå vidare med en bedömning, ett besök eller ett förslag på upplägg.
