När företagsåterbruk blir en smart lösning

Många företag har objekt som inte längre används, men som fortfarande kan vara användbara för någon annan. Det kan handla om äldre maskiner, delar, material, verkstadsutrustning eller objekt som blivit över vid förändringar i verksamheten.

Istället för att allt går direkt till skrot försöker vi bedöma vad som kan återbrukas, återanvändas eller tas tillvara som reservdelar.

Återbruk • Nytta • Cirkulation

Det här kan ofta återbrukas

  • Äldre maskiner
  • Reservdelar
  • Verkstadsutrustning
  • Metall med användningsvärde
  • Blandade objekt från lager eller gård

Fördelarna för företaget

Företagsåterbruk handlar inte bara om miljö. Det handlar också om att frigöra plats, minska bekymmer, skapa bättre ordning och ibland få ut värde ur sådant som annars bara hade setts som skräp.

Varför välja återbruk före att bara kasta

  • Mer kan komma till användning igen
  • Mindre går direkt till avfall
  • Reservdelar och objekt kan få nytt värde
  • Ni frigör plats i verksamheten
  • Det blir en smidigare helhetslösning

Det här hjälper vi företag med

  • Bedömning av återbruksvärde
  • Maskiner och repobjekt
  • Reservdelar och komponenter
  • Metall och material
  • Utrensning där återbruk är fokus
  • Sortering mellan återbruk och skrot

Maskiner

Maskiner som inte längre används men som fortfarande kan ha värde som helhet eller i delar.

Delar

Reservdelar och komponenter som fortfarande kan komma till användning i andra sammanhang.

Material

Metall och utrustning som inte bara behöver ses som avfall, utan som resurser.

Vad vi tittar på först

  • Om objektet har fortsatt användningsvärde
  • Om delar kan tas tillvara
  • Om materialet lämpar sig för återbruk
  • Hur mycket det rör sig om
  • Hur det enklast kan hanteras vidare

Exempel på sådant som ofta går att ta tillvara

  • Äldre men användbara maskiner
  • Delar från repobjekt
  • Verkstadsutrustning som bytts ut
  • Material från överskottslager
  • Objekt som blivit stående men inte är värdelösa
